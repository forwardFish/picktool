# Runtime Artifacts

This directory stores delivery-time evidence required by the agentsystem workflow.

Expected artifact buckets:
- `story_admissions/`: admission packets and scope confirmation
- `story_failures/`: blocked runs and recovery notes
- `story_handoffs/`: builder/tester/reviewer/doc-writer handoffs
- `browser_evidence/`: screenshots, console captures, and route interaction logs
- `qa_results/`: test outputs, fixture notes, and assertion reports
- `traceability_audits/`: Sprint-level requirement-to-evidence audits
- `sprint_acceptance/`: dedicated Sprint Acceptance packets
- `final_acceptance/`: project-wide end-to-end acceptance evidence
- `review_packets/`: code acceptance and acceptance gate packets
